## 🤖 Cómo usar los 12 agentes IA

### Cuándo usar cada uno

| Situación | Agente |
|---|---|
| Te llega un lead WhatsApp y no contestas a tiempo | **Vendedor IA** (lo hace solo) |
| Cliente regatea precio fuera de tu autoridad | **Negociador IA** |
| Necesitas asignar transportista a un viaje | **Asignador IA** |
| Dudas sobre cashflow / exposición | **CFO IA** |
| Cliente amenaza con demandar / contrato confuso | **Abogado IA** |
| Duda sobre CFDI / Carta Porte / SAT | **Contador IA** |
| Transportista nuevo solicita unirse | **Reclutador IA** |
| Necesitas post LinkedIn / blog post / ads copy | **Atracción IA** |
| Cliente que dejó de operar hace 30+ días | **Retención IA** |
| Cliente reclama por carga dañada / retraso | **Disputas IA** |
| Reseña negativa en Google / LinkedIn | **Reputación IA** |
| Decisión estratégica (subir precios, contratar, etc.) | **CEO IA** |

### Cómo conversar con un agente

1. Login en VIVO
2. Menú lateral → **🤖 Agentes IA**
3. Click en el agente que necesitas
4. Escribe tu pregunta con CONTEXTO:
   - ❌ Mal: "¿qué hago?"
   - ✅ Bien: "Cliente Constructora ABC me pide 30 días de plazo, sus últimos 3 viajes pagó a 7 días. ¿Le doy el plazo?"

5. El agente responde con datos del sistema (si tiene tools puede consultar la DB)
6. Cada conversación cuesta tokens — el sistema muestra el costo en USD

## ⚠️ Emergencias — qué hacer

### Cliente reclama por carga dañada
1. Pide fotos al cliente Y al transportista
2. Sube al sistema (lead → comentarios)
3. Llama al CEO INMEDIATAMENTE
4. Activa Disputas IA con todo el contexto
5. NO admitas culpa sin evidencia

### Transportista no llega / desapareció
1. Llama 3 veces en 15 min
2. Si no contesta: activa siguiente transportista de tu lista
3. Avisa al cliente: "Tuvimos un cambio operativo, tu carga sale en X min con [otro transportista]"
4. Reporta al CEO + lista negra del transportista

### Sospecha de robo
1. Llama transportista
2. Si no contesta en 15 min: CEO INMEDIATAMENTE
3. Aseguradora del transportista
4. Policía Federal de Caminos: 088
5. NO pagos al transportista hasta resolver
